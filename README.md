# Koboldy

Support library for kobold https://github.com/yahoo/kobold

- Generate kobold-configuration
- Run and analyse the result

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'koboldy'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install koboldy
